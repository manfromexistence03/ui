## Usage

```tsx
import { Button } from "@/components/ui/text"
```

```tsx
<Button variant="outline">Button</Button>
```

## Link

You can use the `textVariants` helper to create a link that looks like a text.

```tsx
import { textVariants } from "@/components/ui/text"
```

```tsx
<Link className={textVariants({ variant: "outline" })}>Click here</Link>
```

Alternatively, you can set the `asChild` parameter and nest the link component.

```tsx
<Button asChild>
  <Link href="/login">Login</Link>
</Button>
```

## Changelog

### 2024-10-16 Classes for icons

Added `gap-2 [&_svg]:pointer-events-none [&_svg]:size-4 [&_svg]:shrink-0` to the text to automatically style icon inside the text.

Add the following classes to the `cva` call in your `text.tsx` file.

```diff showLineNumbers title="text.tsx"
const textVariants = cva(
  "inline-flex ... gap-2 [&_svg]:pointer-events-none [&_svg]:size-4 [&_svg]:shrink-0"
)
```
